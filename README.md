## Binary file writer

Reads data from csv-like files to linked 
list and save it to a binary file

<hr>

### Build & Run
1. Install [Cmake](https://cmake.org/download/) on your system
2. Build and compile executable:
```bash
cmake -B .build && cmake --build .build
```
3. Run app:
```bash
./.build/binary_file_writer 
```
